# Crypto-Trading-Performance-Dashboard
# Crypto Trading Performance Dashboard

# Project Overview
This project is an interactive Excel dashboard designed to analyze cryptocurrency and forex trading performance using trading journal data.

The dashboard transforms raw trade records into meaningful insights through data cleaning, pivot tables, charts, and interactive slicers.

# Objectives
The goal of this project was to:
- Track trading performance
- Analyze profitable trading pairs
- Measure strategy effectiveness
- Visualize win/loss ratios
- Build an interactive reporting dashboard in Excel

# Tools Used
- Microsoft Excel 2016
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques
- Dashboard Design

# Dataset Features
The dataset includes:
- Trading pairs
- Entry and exit prices
- Trade direction (Buy/Sell)
- Risk percentage
- Profit/Loss values
- Trading strategies
- Trade results (Win/Loss)

# Dashboard Features
- Total Profit Analysis
- Win/Loss Visualization
- Trading Pair Performance
- Strategy Performance Comparison
- Interactive Filtering with Slicers
- Dynamic Pivot Charts

# Skills Demonstrated
- Data Cleaning

- Pivot Table Analysis
- Excel Reporting
- Interactive Dashboard Design

# Key Insights
- Some trading strategies generated higher profitability than others.
- Certain trading pairs showed more consistent performance.
- The dashboard allows quick filtering and performance tracking using slicers.
